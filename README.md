# Lab Simulator

A browser-based liquid handling protocol simulator built with React, TypeScript, and Vite.

## Features

- Drag-and-drop canvas for placing 96-well plates, 384-well plates, and reservoirs
- Protocol builder for defining pipette transfer steps
- Step-by-step simulation with animated well volume tracking
- Protocol import from CSV with an AI-powered review screen
- Stats panel with transfer summary and validation errors
- Export protocols to CSV
- Dark mode support

## Tech Stack

React · TypeScript · Vite · Tailwind CSS
